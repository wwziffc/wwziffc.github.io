这部分代码有一个错误，即在鼠标进入链接时，虽然选中了当前处于 active 状态的链接，但没有给新的链接设置 active 状态。
报错 “Uncaught TypeError: Cannot read properties of null (reading ‘classList’)” 表示代码试图读取一个 null 或 undefined
变量的属性，具体来说，在这个错误中，代码试图读取一个 DOM 元素（通过类名或 ID）的 classList 属性，但是没有找到该元素。

在这个特定的代码中，document.querySelector(‘.tab-nav .active’) 返回了 null，因为此时没有匹配到具有 active 类名的元素，而 null 是没有 classList
属性的，所以代码就会抛出上述错误。

在给链接添加鼠标经过事件的代码中，我们需要操作一个当前处于 active 状态的链接，如果没有找到匹配的元素，就会出现以上的错误。因此，我们需>要确保在操作 DOM 元素之前，它已经被正确地创建并添加到了文档树中。
